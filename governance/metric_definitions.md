# Metric Definitions & Interpretation

| Metric | Definition | Aggregation | Interpretation |
| :--- | :--- | :--- | :--- |
| **Total Revenue** | Sum of `revenue_usd` | SUM | Total financial output generated by products. |
| **Demand Accuracy** | `actual_demand` / `forecast_demand` | AVG | 1.0 = Forecast equals actual; >1.0 = Actual exceeds forecast; <1.0 = Forecast exceeds actual. |
| **Yield Percentage** | Percentage of units meeting quality standards | AVG | Higher values indicate more efficient manufacturing. |
| **Defect Rate** | Defective units divided by total produced units | AVG | Higher values indicate quality risk. |
| **Units Produced** | Total units manufactured | SUM | Production scale indicator. |
| **Throughput Ops** | Operations processed per unit time | AVG | System processing efficiency. |
| **Latency (ms)** | Average processing delay | AVG | System responsiveness. |
